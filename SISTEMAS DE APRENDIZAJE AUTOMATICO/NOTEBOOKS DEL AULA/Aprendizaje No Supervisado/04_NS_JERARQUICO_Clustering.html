Clustering Jerrquico

Importamos las libreras y cargamos el conjunto de datos (Mall_Customers.csv), indicando que la variable que se analizar es una matriz con las columnas 3 y 4 de conjunto de datos, las cuales corresponden al ingreso anual en miles y la puntuacin del cliente.

In[1]:
# Importacion de librerias
import numpy as np
import matplotlib.pyplot as plt
import pandas as pd

# Carga del conjunto de datos
dataset = pd.read_csv('./datasets/Mall_Customers.csv')
X = dataset.iloc[:, [3, 4]].values

Para crear el dendograma utilizamos la clase sch del paquete scipy.hierarchy

In[2]:
# Creamos el dendograma para encontrar el nmero ptimo de clusters

import scipy.cluster.hierarchy as sch
dendrogram = sch.dendrogram(sch.linkage(X, method = 'ward'))

plt.title('Dendograma')
plt.xlabel('Clientes')
plt.ylabel('Distancias Euclidianas')
plt.show()

En el grfico podemos observar que la distancia mxima la marca la lnea azul oscuro que une los clusters rojo y azul claro, por lo que si realizamos el corte en esa zona obtenemos: (En el lado opuesto (el izquierdo), se observa que la distancia ms larga (mxima) se corresponde con la primera lnea del clster naranja. Por tanto el corte debe cortar esa lnea tambin!!)

Al marcar con la lnea color naranja el dendograma donde se observa la distancia mxima, que genera 5 clusters que se han identificado con nmeros.

Con ello, se generan los grupos con el mtodo aglomerante utilizando la clase AgglomerativeClustering del paquete sklearn.cluster:

In[3]:
from sklearn.cluster import AgglomerativeClustering
hc = AgglomerativeClustering(n_clusters = 5, 
                    metric = 'euclidean', 
                    linkage = 'ward')

y_hc = hc.fit_predict(X)

Para poder observar grficamente la asignacin de los 200 clientes a 5 grupos o clusters se plantea lo siguiente:

In[4]:
# Visualizacin de clusters
plt.scatter(X[y_hc == 0, 0], X[y_hc == 0, 1], s = 100, c = 'red', label = 'Cluster 1')
plt.scatter(X[y_hc == 1, 0], X[y_hc == 1, 1], s = 100, c = 'blue', label = 'Cluster 2')
plt.scatter(X[y_hc == 2, 0], X[y_hc == 2, 1], s = 100, c = 'green', label = 'Cluster 3')
plt.scatter(X[y_hc == 3, 0], X[y_hc == 3, 1], s = 100, c = 'cyan', label = 'Cluster 4')
plt.scatter(X[y_hc == 4, 0], X[y_hc == 4, 1], s = 100, c = 'magenta', label = 'Cluster 5')
plt.title('Clusters de clientes')
plt.xlabel('Ingresos anuales (k$)')
plt.ylabel('Puntuacin de gastos (1-100)')
plt.legend()
plt.show()

De nuevo, en relacin al ingreso anual en miles y la puntuacin generada por la tienda, observamos un grupo de clientes que podra ser de inters para la tienda: el grupo de clientes en color verde, lo cuales tienen ingresos altos y una puntuacin alta, por lo que podran ser un grupo objetivo para ciertas promociones.
En purpura tenemos a los clientes de baja puntuacin y bajos ingresos, mientras que en azul claro, a los clientes con ingresos bajos pero con alta puntuacin, lo cual podra indicar que estos clientes compran mucho a pesar de los ingresos bajos.
